Introduction
============
The menu template is by AdminLTE but the Javascript to make the menu work without refresh (choose option and get an article has been written by Akhil Gupta).

This makes the sidemenu completely reusable and anyone can use it.

All you need to do is, create the option in the index file and add the content in the .assets/includes/ folder.
